Graded elements present in this Project:

  • CRAP Techniques. Kept the styling consistent throughout the entire website.

  • Organization of information. Each piece of information is in its own identifiable section container. It has its own subheader and its own background color.

  • Content. Including a map that specifies the location of the place.

  • Sufficient pages of content.

  • Code Design Requirements. Indentation of code, single global CSS file, use of semantic tags, no inline styles used.

  • User friendly website. Has global navigation, a unique title tag for each page, and the page you are on is highlighted on the navigation.

  • Validation for all HTML & CSS.

  • Responsiveness for Desktop and Mobile displays.
